# 网络安全工具

## 原始项目地址

[Hack-Tools - Web 渗透的一体化红队工具](https://github.com/LasCC/Hack-Tools)

[peAssist - Windows 提权辅助](https://github.com/gh0stkey/peAssist)

[xssee - 一个用于 XSS 编码、解码的简单工具](https://github.com/Monyer/xssee)

[unicodejsp - Unicode 编码 JSP 代码](https://github.com/Rvn0xsy/usefull-code)

[GoogleHackingTool - 在线 Google Hacking 小工具](https://github.com/r00tSe7en/GoogleHackingTool)

[avList - 杀软进程对应杀软名称](https://github.com/gh0stkey/avList)

[get_AV](https://github.com/r00tSe7en/get_AV)

[CyberChef - 用于加密、编码、压缩和数据分析的网站](https://github.com/gchq/CyberChef)

[reverse-shell-generator - 功能强大的反弹 Shell 生成器](https://github.com/0dayCTF/reverse-shell-generator)
